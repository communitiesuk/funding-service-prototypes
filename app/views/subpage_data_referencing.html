Data referencing journey

This functionality allows a form designer to reference data that has previously been uploaded to the service as a CSV.

The prototype illustrates:

  • adding a condition (without a calculation)
  • referencing data in a data set that has been uploaded
  • selecting a column from the data set
  • defining the condition (greater than, less than or between)
  • saving the condition

This journey has not yet been tested with users but will be tested as part of upcoming rounds.
